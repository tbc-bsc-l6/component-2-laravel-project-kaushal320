Summary and Reflection on Artificial Intelligence (AI) Usage

This section documents the primary AI prompts used during the development of this assignment. Each prompt contributed to understanding technical concepts, resolving implementation challenges, and improving the overall quality of the final solution.

Prompt: How can Google OAuth authentication be implemented in Laravel using Socialite for local development?

Date: 28-12-2025

AI Response Summary: The AI provided a structured explanation of configuring Laravel Socialite with Google OAuth, including environment setup, redirect and callback handling, and user authentication flow.

Reflection: This response clarified the OAuth authentication process and reduced configuration errors. It enabled a secure and reliable social login implementation in the application.

Prompt: How can Ollama AI be integrated with Laravel to create a real-time streaming chat feature?

Date: 03-01-2026

AI Response Summary: The AI explained how to use Laravel’s streaming responses to process partial outputs from the Ollama API and deliver them to the frontend in real time.

Reflection: This prompt improved my understanding of streaming data between backend and frontend. It enabled the creation of a responsive chat interface with incremental AI responses.

Prompt: How can markdown-style formatting be handled in React chat messages?

Date: 07-01-2026

AI Response Summary: The AI described methods for detecting markdown-like syntax using regular expressions and dynamically converting formatted text into JSX elements.

Reflection: This enhanced the readability of AI-generated content and strengthened my skills in text parsing and conditional rendering within React.

Prompt: How can Laravel Fortify login be customized to work with Inertia.js?

Date: 11-01-2026
